HTML

  • HTML gives content structure
  • <!DOCTYPE html>: declaration tells browser to expect HTML on page
  • <html>: opening tag at start of doc
    • <html lang="en">: attribute tells browser lang for webpage is English
  • The head element contains information about the webpage.
    • <title>: title for tab of webpage
    • <style>: defines CSS styles associated with website
    • <base>: base URL for webpage
    • <link>: connects external resource to HTML doc
    • <meta>: metadata (i.e. char encoding, keywords, author)
      • charset: sets character encoding
      • http-equiv: sets HTTP header for content
      • viewport: sets default view for webpage
  • The body element represents the visible content shown to the user.
    • <h#>: creates diff sizes/levels of headers/subtitles (1 is largest)
    • <p>: paragraph/block of text
    • Lists
      • <ul>: unordered lists
      • <ol>: ordered lists
      • <li>: list items
    • <img>: info about images
      • src: location of image
      • alt: text that describes appearance & functionality of img
    • <br>: creates line of empty space
    • <a>: creates hyperlink to same or other webpage
      • href: points to URL for link
    • <footer>: author, copyright, contact, sitemap, navigation, etc.

CSS

  • CSS gives style, aesthetics, organization
  • 3 ways:
    • Inline CSS: to style element directly in HTML file (use style attribute inside HTML tag)
    • Internal CSS Style Sheet: to embed entire stylesheet directly in HTML file using <style> element in <head> element
    • External CSS Style Sheet: to keep CSS rules in separate file, use <link> inside <head> to link file
  • CSS rule:
    • Selector defines element or attributes to which rules apply
      • * = all elements on page
    • Declaration includes CSS property and value
    • Class attribute = share CSS rule to any element by assigning the rule to class attribute with class selector
      • ".class" means style applies to all elements assigned to this class
  • CSS box model:
    • Margin: space around outside of element
    • Padding: space around content inside element
  • <img> selector
    • "display: block": assigns each image on page to start on new line

Git

  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches to existing branch
  • git status: checks what branch we are currently on
  • git add -A: adds all changes in current working branch to staging area
  • git commit -m "message": commit files with a message about the changes
  • git pull origin main: pull latest version of main branch to local computer
  • git push origin branch-name: push changes in local branch to remote GitHub branch

JavaScript

  • JavaScript(JS) adds interactivity to website
  • Variable is a named container that allows us to store data in our code.
    • Use "var variable_name = " to create variable and assign data to it
  • "console.log()": output message to web console
    • Open Chrome DevTools: Ctrl+Shift+I
  • JS file linked within <body> towards the end
    • "<script src="./assets/script.js"></script>"
  • Control flow is the order in which a computer executes code in a script.
  • Conditional Statements:
    • Falsy: 0, -0, On, "", null, undefined, NaN, Boolean with value of false
    • Truthy: all other values, Boolean with value of true
    • === : checks to see if two values are equal, returns Boolean result (true or false)
    • Use conditional statements (i.e. if-else statements) to interrupt control flow
  • Arrays:
    • Declaration: "var array_name = [...]"
    • Can store multiple data items (of same or diff types) in variable
    • Index: number that identifies location of element in array
    • First element in array has index of 0
    • "array_name[index_number]" returns the element in array at that position
    • "array_name.length" returns # of elements in array
  • For-loop:
    • Performs task on all items in array by allowing single code block to be executed over and over
    • for(starting point; condition; iteration){ ... }
  • Functions:
    • Function: set of instructions that tells computer how to perform certain task
    • Declaration: "function name(parameters){ ... }"
    • Need to call function in script to execute it: "function_name(insert any arguments);"
