HTML

  • HTML gives content structure
  • <!DOCTYPE html>: declaration tells browser to expect HTML on page
  • <html>: opening tag at start of doc
    • <html lang="en">: attribute tells browser lang for webpage is English
  • The head element contains information about the webpage.
    • <title>: title for tab of webpage
    • <style>: defines CSS styles associated with website
    • <base>: base URL for webpage
    • <link>: connects external resource to HTML doc
    • <meta>: metadata (i.e. char encoding, keywords, author)
      • charset: sets character encoding
      • http-equiv: sets HTTP header for content
      • viewport: sets default view for webpage
  • The body element represents the visible content shown to the user.
    • <h#>: creates diff sizes/levels of headers/subtitles (1 is largest)
    • <p>: paragraph/block of text
    • Lists
      • <ul>: unordered lists
      • <ol>: ordered lists
      • <li>: list items
    • <img>: info about images
      • src: location of image
      • alt: text that describes appearance & functionality of img
    • <br>: creates line of empty space
    • <a>: creates hyperlink to same or other webpage
      • href: points to URL for link
    • <footer>: author, copyright, contact, sitemap, navigation, etc.

Git

  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches to existing branch
  • git status: checks what branch we are currently on
  • git add -A: adds all changes in current working branch to staging area
  • git commit -m "message": commit files with a message about the changes
  • git pull origin main: pull latest version of main branch to local computer
  • git push origin branch-name: push changes in local branch to remote GitHub branch
